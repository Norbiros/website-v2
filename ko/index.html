All-in-one browser extension for Scratch

스크래치 애드온은 스크래치 웹 사이트와 프로젝트 편집기에 새로운 기능과 테마를 제공하는 브라우저 확장 기능입니다. 스크래치 커뮤니티의 여러 구성원이 개발한 사용자 스타일과 사용자 스크립트를 수집, 보관 및 사용하고 있습니다.

기능

애드온

스크래치 애드온은 스크래치 웹 사이트와 편집기를 더욱 편하게 사용할 수 있는 기능인 추가 기능으로 구성됩니다.

사용자 지정 설정하기

스크래치 애드온에는 각 기능과 테마의 다양한 설정을 제어하는 사용하기 쉬운 인터페이스가 있습니다.

국제적

영어뿐만이 아닙니다. 스크래치 애드온은 한국어를 포함해 스페인어, 프랑스어와 더 많은 언어로 사용하실 수 있습니다.

오픈소스

스크래치 애드온은 무료이며 오픈소스이고 스크래치 사용자들에 의해 만들어졌습니다.

주요 기능

기억하세요, 이것은 주요 기능일 뿐입니다. 이 외에도 정말 많은 기능이 존재합니다. 이곳에서 더 알아보세요.

  • 60FPS 프로젝트 플레이어 모드

    Alt+깃발 클릭하여 60fps를 토글합니다.

  • 썸네일 설정기

    편집기의 오른쪽 모퉁이에 메뉴바를 추가하여 프로젝트의 썸네일을 모든 이미지(gif 포함)로 설정합니다.

  • 블럭 바꾸기

    블럭을 마우스 오른쪽 버튼으로 클릭하여 관련 블럭으로 전환합니다.

  • 복제본 개수 표시

    전체 복제본의 수를 편집기 무대 상단에 표시합니다.

  • 클라우드 게임

    스크래치 애드온 아이콘을 클릭할 때 사용할 수 있습니다. 클라우드 게임에 현재 얼마나 많은 스크래쳐가 있는지와 해당 사용자 이름을 표시합니다.

  • HEX 색상 선택기

    색상 선택기에 HEX 코드 입력을 추가합니다.

  • 확인

    프로젝트 공유와 같은 스크래치 계정과 관련한 작업을 수행할 때 한 번 더 묻습니다.

  • 댓글 버튼에 복사 링크 추가

    모든 웹사이트 댓글에서 "신고하기" 버튼 옆에 "링크 복사" 버튼을 추가합니다.

  • 데이터 카테고리 개선

    블록 카테고리에서 데이터("변수") 카테고리를 개선합니다.

  • 파일 끌어다 놓기

    파일 탐색기에서 이미지, 소리를 스프라이트 창이나 의상/사운드 목록으로 끌어다 놓을 수 있습니다. 또한 텍스트 파일을 목록으로 끌어다 놓거나 "묻고 기다리기" 질문창을 표시할 수 있습니다.

  • 개발자 도구

    Adds a find box and new menu options to the editor: copy/paste blocks, better clean up, go to custom block definition, and more! Press Ctrl+F to focus the find box, Ctrl+Space to bring up the "add block by name" popup, Ctrl+Left/Right to navigate find history.

  • 편집기 내 메시지

    편집기 메뉴 바에서 메시지 수를 보여줍니다.

  • 검색 가능한 드롭다운

    블록 드롭다운을 검색할 수 있도록 합니다.

  • 정확한 정보 표시

    사용자 및 스튜디오 정보의 정확한 정보를 보입니다. 토론방 정보는 ScratchDB를 사용합니다.

  • 더 나은 편집기 주석

    스크래치 편집기에서 주석 관련 기능을 변경합니다. 블록에 달린 주석이 드래그한 후 위치가 올바르게 저장되지 않는 버그를 수정하고, 블록의 주석이 큰 스크립트에서 화면 밖으로 생성되지 않도록 방지하고, 블록 주석 연결을 그대로 유지하는 등의 작업을 수행합니다.

  • 블럭을 복제한 뒤 자동으로 실행되는 버그 고치기

    스크래치 편집기에서 복제 뒤 블럭이 다른 블럭에 붙을 경우 실행되는 버그를 고칩니다.

  • 스프라이트 폴더

    Adds folders to the sprite pane, as well as costume and sound lists. To create a folder, right click any sprite and click "create folder". Click a folder to open or close it. Right click a sprite to see what folders you can move it to, or alternatively drag and drop it into an open folder. This feature works by adding "[folderName]//" at the beginning of the names for your sprites.

  • 전체 구역

    토론방의 서명이나 프로필에서의 내가 하고 있는 일과 같은 높이가 제한되어 있는 기능을 크기를 조정할 수 있게 하거나 스크롤 가능하게 해 전체 내용을 보입니다.

  • 블록 팔레트 자동으로 숨기기

    사용되지 않은 경우 블록 팔레트를 숨깁니다. 잠금 아이콘을 클릭하여 일시적으로 제자리에 잠그거나 또는 "카테고리 클릭" 모드를 사용하세요.

  • 프로젝트 영상 녹화기

    편집기 메뉴 바에 "녹화 시작" 버튼을 추가해 프로젝트의 무대를 녹화할 수 있도록 합니다.

  • 마우스 위치 표시

    마우스의 x/y위치를 편집기 무대의 상단에 표시합니다.

  • 스튜디오 아이콘에서 메시지 개수 표기하기

    스크래치 애드온 아이콘에 받은 메시지 수를 추가합니다.

  • 프로젝트 플레이어 음소거 모드

    녹색 깃발을 Ctrl+클릭하여 프로젝트를 음소거/음소거 해제합니다.

  • 어니언 스키닝

    모양을 편집할 때 전 모양이나 다음 모양을 투명한 오버레이로 보입니다. 모양 편집기에서 확대 버튼 옆의 버튼으로 동작합니다.

  • 모양을 기본적으로 채우기

    "스프라이트/모양/배경/소리 고르기" 버튼의 기본 동작을 변경합니다.

  • 일시정지 버튼

    초록 깃발 옆에 일시 정지 버튼을 추가합니다.

  • 프로젝트 진행 표시줄

    프로젝트를 불러오거나 저장할 때 진행 표시줄을 추가합니다.

  • 리믹스 트리 버튼

    프로젝트 페이지의 링크 복사하지 옆에 다시 리믹스 버튼을 표시합니다.

  • 스프라이트 제거 확인

    스프라이트를 지울 때 한 번 더 묻습니다.

  • 크기 조절 가능한 댓글 입력창

    스크래치 3.0 스타일 페이지에서 댓글 입력 상자의 크기를 상하로 조정합니다.

  • 스크래치 메시지

    스크래치 애드온 아이콘을 클릭할 때 사용할 수 있습니다. 스크래치 메시지를 쉽게 읽고 답장할 수 있습니다. 메시지를 프로젝트별로 그룹화하고, 댓글과 답글을 표시하며, 직접 메시지를 답장할 수 있습니다.

  • 스크래치 알리미

    새 스크래치 메시지에 알리미입니다. 알림을 클릭하여 메시지(댓글, 프로젝트 등)로 바로 이동합니다.

  • 스크래치 2.0 → 3.0

    스크래치 2.0 형식의 페이지를 스크래치 3.0처럼 보이게 합니다.

  • 프로필 찾기

    사용자 이름을 검색할 때 사용자의 프로필로 이동할 수 있습니다.

  • 스튜디오 도구

    "큐래이터"탭에 쉽게 메니저를 승급하거나 제거할 수 있게 합니다. 그리고 "스튜디오 팔로우하기"버튼 아래에 스튜디오를 나갈 수 있는 버튼을 추가합니다.
