Alternativen

Falls du eine Alternative brauchst, gibt es hier ein paar Möglichkeiten.

  • Erstelle ein Issue auf dem Repository.
  • Erstelle ein Post in unserem Diskussionstab
  • Sende eine Nachricht auf unserem Discord-Server
  • Kontaktiere einen unserer Entwickler auf Discord, Twitter, oder anderen Methoden

Wegen der Richtlinie, die das Werben von Erweiterungen/Userscripts verbietet ist es nicht empfohlen, einen der Entwickler auf Scratch zu kontaktieren.
