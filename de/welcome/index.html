Willkommen!

Scratch Addons wurde erfolgreich installiert!

Danke fürs Installieren. Wir hoffen, dass dir die Erweiterung gefallen wird!

Erste Schritte

Schritt 1
Klicke das Icon der Scratch Addons-Erweiterung an

Es befindet sich rechts oben in deinem Browser. Nutzer von Google Chrome müssen auf das Puzzleicon klicken, um das Icon zu finden.

Schritt 2
Drücke auf das Zahnradsymbol, um zu den Einstellungen zu gehen

Dadurch wird die Einstellungenseite geöffnet, wo du die Funktionen der Erweiterung konfigurieren kannst.

Schritt 3
Scrolle durch die Addonsliste und aktiviere die, die du willst

Du kannst auch jedes Addons in dessen unteren Bereich einzeln konfigurieren.

Willst du uns untertützen?

Du kannst einen Stern auf unserem Repository auf GitHub vergeben oder eine Bewertung im Chrome Web Store oder im Firefox Add-ons Store hinterlassen.
