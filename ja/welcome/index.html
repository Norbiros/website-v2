Scratch Addonsがインストールされました!

インストールありがとうございます。ぜひ、この拡張機能をお楽しみください。

始め方

ステップ 1
Scratch Addons拡張機能アイコンをクリック

ブラウザーの右上にあります。Google Chromeでは、パズルのアイコンをクリックしてください。

ステップ 2
歯車アイコンで設定ページに行く

設定ページが開きます。ここで、拡張機能を設定できます。

ステップ 3
アドオンのリストから、使いたいものを有効化する

アドオンには、さらに細かく設定できるものもあります。

支援したいですか?

GitHub リポジトリにスターしたり、Chrome 拡張機能ページやFirefox アドオンページにてレビューを投稿できます。
