Dependencies

These dependencies are being used to make Scratch Addons possible.

Extensión

  • scratchblocks (Apache-2.0)
  • Vue (BSD-3-Clause)
  • ICU Message Formatter (Apache-2.0)
  • TinyColor (MIT)
  • text-field-edit (MIT)
  • Chart.js (MIT)
  • color-picker-web-component (MIT)
  • Comlink (Apache-2.0)
  • Sora (OFL)
  • Fuse.js (Apache-2.0)
  • idb (ISC)
  • Sortable.js (MIT)
  • spark-md5 (MIT)

Sitio Web

  • Hugo (Apache-2.0)
  • Bootstrap (MIT)
  • Popper (MIT)
  • jQuery (MIT)
  • Iconify (Apache 2.0)
