Список всех аддоров, которые созданы в Scratch Addons.

139 аддонов

2D-палитра цветов
2d-color-picker

Replaces saturation and brightness sliders in the costume editor with a 2D color picker. Hold Shift while dragging the cursor to change the values on a single axis.

60FPS project player mode 🌟

Alt+Клик по зелёному флагу, для включения 60FPS.

Аддон, который ничего не делает 🥚

Буквально. Аддон который ничего не делает, но в других языках исправляет капитализацию у «Account settings» на «Account Settings».

Установщик обложек 🌟

Улучшенные цитаты для форумов

Отладчик ⭐
debugger

Добавляет новое окно «отладчика» к редактору. Отладчик позволяет логировать в вкладку «Логи» окна отладчика используя блоки «логировать», «предупредить» и «ошибка». Блок «точки остановки» поставит проект на паузу при выполнении. Все выполняемые потоки могут быть просмотрены в вкладке «Потоки» отладчика, и при остановка кнопка «Шаг вперёд» может быть использован для выполнения следующего блока. График кадровой частоты (FPS) и кол-во клонов могут быть просмотрены в вкладке «Производительность».
