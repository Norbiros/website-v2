Расширение «всё в одном» для браузера для Scratch

Scratch Addons — расширение для браузера, которое даёт новые возможности для веб сайта Scratch и его редактора проектов. Его миссия — коллекционировать, архивировать и делать юзерстили и юзерскрипты используемыми, которые разработаны некоторыми участниками сообщества Scratch.

Особенности

Аддоны

Scratch Addons содержит аддоны — возможности, которые могут расширить Ваше впечатление веб сайта Scratch и редактора.

Настраиваемый

У Scratch Addons есть интерфейс, который лёгкий в использовании, чтобы изменять различные настройки каждой возможности и темы.

На разных языках

Не только на английском. Scratch Addons есть на испанском, русском, японском, французском и на многих других языках.

Открытый исходный код

Scratch Addons бесплатный, с открытым исходным кодом и разработан сообществом Скретчеров.

Основное

Это всего-лишь самое основное. Есть ещё!

  • 60FPS project player mode

    Alt+Клик по зелёному флагу, для включения 60FPS.

  • Установщик обложек

    Adds a button to the bottom-right of the project page to set the thumbnail of your project to any image (including gifs).

  • Замена блоков

    Щёлкните правой кнопкой мыши по блоку, чтобы заменить его на связанный блок.

  • Счётчик клонов

    Adds a counter above the stage in the editor which shows the total amount of clones.

  • Облачные игры

    Available when clicking the Scratch Addons icon. Shows you how many Scratchers are currently on multiplayer projects, and their usernames.

  • Выбор цветов типа Hex

    Добавляет ввод кодов hex в выбор цветов.

  • Подтверждения

    Спрашивает, уверены ли вы, когда совершаете действия под своим аккаунтом Scratch, например, делитесь проектами.

  • Кнопка «скопировать ссылку на комментарий»

    Добавляет кнопку "Копировать ссылку" ко всем комментариям на сайте рядом с кнопкой "Пожаловаться".

  • Поправки категории данных

    Provides tweaks for the Data ("Variables") block category.

  • Перетаскивание файлов

    Lets you drag images and sounds from your file manager into the sprite pane or costume/sound list. You can also drag text files into lists or "ask and wait" question inputs.

  • Инструменты разработчика

    Adds a find box and new menu options to the editor: copy/paste blocks, better clean up, go to custom block definition, and more! Press Ctrl+F to focus the find box, Ctrl+Space to bring up the "add block by name" popup, Ctrl+Left/Right to navigate find history.

  • Сообщения в редакторе

    Displays your message count in the editor menu bar.

  • Поиск в выпадающих списках

    Позволяет искать в раскрывающемся списке блоков.

  • Показывать точный счёт

    Показывает точный счёт для информации о пользователе и студии. Использует ScratchDB для информации о форуме.

  • Улучшенный редактор комментариев

    Делает многие изменения к комментариям в редакторе Scratch: исправляет баг, где комментарии, соединённые к блокам не сохраняют их позицию правильно после перемещения, запрещает комментариям блоков появляться вне экрана в больших скриптах, держит прямыми соединения комментариев с блоками и др.

  • Не выполнять автоматически дублированные блоки

    Исправляет ошибку в редакторе Scratch, когда дублирование блока и присоединение его к стопке приводит к запуску стопки.

  • Папки спрайтов

    Adds folders to the sprite pane, as well as costume and sound lists. To create a folder, right click any sprite and click "create folder". Click a folder to open or close it. Right click a sprite to see what folders you can move it to, or alternatively drag and drop it into an open folder. This feature works by adding "[folderName]//" at the beginning of the names for your sprites.

  • Full areas

    Shows the full content of areas with limited height, such as user signatures on the forums and What I've been doing on profiles, by making them resizable or scrollable.

  • Автоскрытие палитры блоков

    Скрывает палитру блоков, если не наведено. Щелкните значок замкА, чтобы временно заблокировать его. Как вариант, используйте режим «щелчок по категории».

  • Запись проекта на видео

    Добавляет кнопку "начать запись" в меню редактора, которая позволяет записывать сцену проекта.

  • Позиция мыши

    Displays your mouse x/y position above the stage in the editor.

  • Количество сообщений на значке расширения

    Добавляет текущее количество сообщений на значок Scratch Addons.

  • Беззвучный режим проектов

    Клавиша Ctrl + Клик по зелёному флагу чтобы выключить/включить звук проекта.

  • Отображение Onion

    Shows transparent overlays of previous or next costumes when editing a costume. Controlled by buttons under the costume editor by the zoom buttons.

  • Рисовать костюм по умолчанию

    Изменяет действие по умолчанию кнопок «выберите спрайт/костюм/фон/звук», которые по умолчанию открывают библиотеку.

  • Кнопка паузы

    Добавляет кнопку паузы рядом с зелёным флагом.

  • Индикатор загрузки проекта

    Добавляет индикатор загрузки и сохранения проектов.

  • Кнопка дерева ремиксов

    Возвращает кнопку дерева ремиксов на страницы проектов рядом с кнопкой Копировать ссылку.

  • Sprite deletion confirmation

    Спрашивает, уверены ли вы при удалении спрайта внутри проекта.

  • Resizable comment input

    Makes the comment input box vertically resizable in Scratch 3.0-styled pages.

  • Сообщения Scratch

    Available when clicking the Scratch Addons icon. Provides easy reading and replying to your Scratch messages: groups messages by project, shows full comment text and context, allows direct comment replying.

  • Уведомления Scratch

    Notifications on new Scratch messages. Click the notification to go straight to the message (comment, project, etc.)

  • Scratch 2.0 → 3.0

    Делает страницы в стиле Scratch 2.0 похожими на Scratch 3.0.

  • Поиск профиля

    Позволяет перейти к профилю пользователя при поиске его имени пользователя.

  • Studio tools

    Lets managers easily promote curators to managers or remove anyone from a studio on the "curators" tab, and adds the ability to leave a studio using a button under "Follow Studio".
